Football Badge Memory Game

Game Description

This is a memory game. The aim is to match all the Premier League Club badges. Match the club badges the quickest and earn yourself top spot on the leaderboard. The person with the quickest time will WIN... nothing it just for fun!

How To Play

  1. Enter your name, click start and the timer will begin.
  2. Click each Premier League Logo to reveal a clubs badge.
  3. Match the club badges up together.
  4. When you've matched the last club badges together the timer will stop.
