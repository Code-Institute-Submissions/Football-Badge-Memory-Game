Football Badge Memory Game

Game Description

This is a simple memory game. The aim of the game is match all the cards. Match the cards the quickest and earn yourself a place on the leaderboard. The person with the quickest time will WIN... nothing it just for fun!

The Rules

  1. To start the game click the start button and the timer will start.
  2. Try to match the cards as quick as you can.
  3. When you match the last cards the timer will stop.
